# The Judy & Ron Endeman Travel Legacy
### "The Contagious Traveling Bug!"

An interactive web-based travel memorial for **Ron & Judy Endeman** (89 years old), built to be presented at their **Legacy Evening** alongside a Shutterfly photo book. Ron has taken 20 coworkers and his family on annual trips for 10+ years — all 7 continents.

---

## Project Status (as of March 2026)

### ✅ Done
- Full NatGeo-editorial dark design (Bebas Neue + Crimson Text, gold accents)
- Interactive Leaflet map — **Esri World Dark Gray** (dark theme, guaranteed English labels)
  - Flight paths with Bezier arcs + **cruise ship routes** (dotted sea-level curves for European Cruise & Mediterranean)
- 12 trip cards with continent filters — corrected to "Australia & the Pacific" (not Oceania)
- Masonry photo gallery with lightbox + GPS coordinates (English dates)
- Stats section with animated counters (7 continents, 30 countries, 50 destinations)
- Photo orientation fix (EXIF transpose applied to all 293 thumbnails)
- Scanner (`scripts/scan_photos.py`) — Desktop JPEG folders + Apple Photos Library (osxphotos)
- 293 photos scanned: Egypt (81), Africa (57), Asia (43), Australia/NZ (28), Mediterranean (24), Central America (23), European Cruise (12), Antarctica (11), Baja Mexico (10), Alaska (4)
- **Year Timeline** — horizontal scrollable bar showing all 12 trips chronologically

  - **Act 1:** Tribute screen (Ron & Judy dedication)
  - **Act 2:** Auto-flying map journey — flies to each trip, highlights paths, shows geo-photo pins
  - **Act 3:** Gallery finale with photo mosaic + closing message
  - Background music: "Come Fly With Me" by Frank Sinatra (fades in/out)
  - Keyboard controls: `SPACE` pause · `→` next · `←` prev · `ESC` exit · `F` fullscreen

## Architecture

```
Viajes_Endeman/
├── index.html              Main single-page app
├── css/style.css           NatGeo dark editorial theme
├── js/
│   ├── data.js             All trip data (12 trips, 50+ destinations, years, coordinates)
│   ├── map.js              Leaflet map, flight paths, cruise routes, presentation highlights
│   ├── timeline.js         Trip cards grid + modal (ModalModule)
│   ├── gallery.js          Masonry photo gallery + lightbox (loads photos.json)
│   ├── presentation.js     Cinematic presentation mode (3 acts, music, keyboard nav)
│   └── app.js              Main init, counters, navbar, year timeline
├── audio/
│   └── come-fly-with-me.m4a   Frank Sinatra — presentation background music
├── data/
│   └── photos.json         Generated by scanner — 293 photos currently
├── photos/
│   ├── africa/             Symlinks to AFRICA1 + AFRICA_2 on Desktop
│   ├── egypt/              Symlinks to EGYPT on Desktop
│   ├── asia/               Symlinks to ASIA on Desktop
│   ├── australia_nz/       Symlinks to AUSTRALIA_&_NEW_ZELAND_ on Desktop
│   ├── mediterranean/      Symlinks to ISRAEL on Desktop
│   ├── central_america/    Symlinks to PANAMA TO PARADISE on Desktop
│   ├── european_cruise/    Symlinks to European Cruise on Desktop
│   ├── antarctica/         Symlinks to ANTARTICA on Desktop
│   ├── baja_mexico/        Symlinks to BAJA on Desktop
│   └── alaska/             Symlinks to ALASKA on Desktop
├── destinations.csv        City list for review/correction
└── scripts/
    ├── scan_photos.py      Photo scanner (Pillow + osxphotos 0.75+)
    └── fix_orientation.py  Force-regenerate thumbnails with correct EXIF rotation
```

**No build tools. No framework. No API keys required.** Pure HTML/CSS/JS + Leaflet (CDN).

## Presentation Mode Controls

| Key | Action |
|-----|--------|
| `SPACE` | Pause / resume (or skip tribute in Act 1) |
| `→` | Skip to next trip |
| `←` | Go to previous trip |
| `ESC` | Exit presentation |
| `F` | Toggle fullscreen |
| `P` (when not active) | Start presentation |
